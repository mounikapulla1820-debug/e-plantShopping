e-plantShopping is a React-based web application developed as a final project submission.  
The application simulates an online plant nursery where users can browse plants, view categories, and add plants to a shopping cart.

## Features
- Landing page for Paradise Nursery with background image
- Product listing page with multiple plant categories
- Each plant displays an image, name, and price
- Add to Cart functionality using Redux
- Dynamic cart item count in the navigation bar
- Shopping cart page with quantity increase/decrease and remove options
- Total price calculation for items in the cart
- Continue shopping and checkout (Coming Soon)

## Technologies Used
- React.js
- Redux Toolkit
- JavaScript (ES6)
- HTML & CSS

## Project Purpose
This project was developed to demonstrate understanding of React components, state management using Redux, routing, and UI development.
